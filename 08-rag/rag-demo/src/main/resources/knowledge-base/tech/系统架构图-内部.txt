## 服务间通信

```
┌──────────────┐     gRPC      ┌──────────────┐
│  服务 A      │ ────────────▶ │  服务 B      │   同步调用
└──────────────┘               └──────────────┘

┌──────────────┐     Kafka     ┌──────────────┐
│  服务 A      │ ────────────▶ │  服务 B      │   异步消息
└──────────────┘               └──────────────┘
```

## 部署架构

- **开发环境**: 单机 Docker Compose
- **测试环境**: 3节点 K8s 集群
- **生产环境**: 多可用区 K8s 集群 (3主6从)

## 性能指标

- QPS: 10,000+
- 响应时间: P99 < 100ms
- 可用性: 99.9%
