### 关键代码片段

**SyncController.java** - 入口
```java
@PostMapping("/trigger")
public ResponseEntity<?> triggerSync(
        @RequestParam String source,
        @RequestParam(defaultValue = "incremental") String mode) {

    // 找到对应的知识源
    KnowledgeSource knowledgeSource = findSource(source);

    // 调用 Pipeline 执行同步
    boolean fullSync = "full".equalsIgnoreCase(mode);
    KnowledgePipeline.SyncResult result = pipeline.sync(knowledgeSource, fullSync);

    return ResponseEntity.ok(Map.of(
        "success", true,
        "docCount", result.docCount(),
        "segmentCount", result.segmentCount()
    ));
}
```

**KnowledgePipeline.java** - 核心处理
```java
public int processDocument(KnowledgeDocument doc) {
    // 1. 删除旧向量
    vectorStore.deleteByDocId(doc.getDocId());

    // 2. 构建 Document
    Metadata metadata = new Metadata();
    metadata.put("docId", doc.getDocId());
    metadata.put("title", doc.getTitle());
    Document document = Document.from(doc.getContent(), metadata);

    // 3. 分割文档
    List<TextSegment> segments = getSplitter().split(document);

    // 4. 批量向量化并存储
    for (int i = 0; i < segments.size(); i += embeddingBatchSize) {
        List<TextSegment> batch = segments.subList(i, end);

        // 向量化
        List<Embedding> embeddings = new ArrayList<>();
        for (TextSegment segment : batch) {
            Embedding embedding = embeddingModel.embed(segment.text()).content();
            embeddings.add(embedding);
        }

        // 存储到向量库
        vectorStore.storeBatch(embeddings, batch);
    }

    return segments.size();
}
```

### 关键代码片段

**SearchController.java**
```java
@GetMapping("/search")
public ResponseEntity<?> search(
        @RequestParam String q,
        @RequestParam(defaultValue = "5") int limit,
        @RequestParam(defaultValue = "0.5") double minScore,
        @RequestParam(required = false) String category) {

    // 调用搜索服务
    List<SearchResult> results = searchService.search(q, limit, minScore, category);

    return ResponseEntity.ok(Map.of(
        "query", q,
        "results", results
    ));
}
```

**KnowledgeSearchService.java**
```java
public List<SearchResult> search(String query, int maxResults,
                                  double minScore, String category) {
    // 1. 把查询文本转成向量
    Embedding queryEmbedding = embeddingModel.embed(query).content();

    // 2. 在向量库中搜索相似向量
    List<EmbeddingMatch<TextSegment>> matches =
        vectorStore.search(queryEmbedding, maxResults, minScore, category);

    // 3. 转换成返回结果
    return matches.stream()
        .map(match -> new SearchResult(
            match.embedded().metadata().getString("title"),
            match.embedded().text(),
            match.score(),
            match.embedded().metadata().getString("category")
        ))
        .toList();
}
```

**RedisVectorStoreService.java**
```java
public List<EmbeddingMatch<TextSegment>> search(
        Embedding queryEmbedding, int maxResults, double minScore) {

    // 构建搜索请求
    EmbeddingSearchRequest request = EmbeddingSearchRequest.builder()
            .queryEmbedding(queryEmbedding)  // 查询向量
            .maxResults(maxResults)           // 最多返回几个
            .minScore(minScore)               // 最低相似度
            .build();

    // 执行搜索 (Redis 内部用 KNN 算法找最近的向量)
    EmbeddingSearchResult<TextSegment> result = store.search(request);

    return result.matches();
}
```

## 阅读代码顺序建议

如果你想理解整个项目，建议按这个顺序阅读:

1. **先看接口** (理解抽象)
   - `KnowledgeSource.java` - 知识源接口
   - `VectorStoreService.java` - 向量存储接口

2. **再看实现** (理解具体)
   - `DatabaseKnowledgeSource.java`
   - `RedisVectorStoreService.java`

3. **看核心流程**
   - `KnowledgePipeline.java` - 同步流程
   - `KnowledgeSearchService.java` - 搜索流程

4. **看入口**
   - `SearchController.java`
   - `SyncController.java`

5. **看配置**
   - `application.yml` - 理解各种配置项
   - `EmbeddingModelConfig.java` - 理解 Bean 如何创建
