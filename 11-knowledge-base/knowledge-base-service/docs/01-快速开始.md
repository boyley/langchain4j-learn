# 快速开始指南

## 环境要求

- JDK 17+
- Maven 3.8+
- Docker (用于运行 MySQL 和 Redis)
- OpenAI API Key

## Step 1: 启动基础设施

```bash
cd 11-knowledge-base/knowledge-base-service

# 启动 MySQL 和 Redis
docker-compose up -d mysql redis-stack

# 验证启动成功
docker ps
# 应该看到 knowledge-mysql 和 knowledge-redis 两个容器
```

## Step 2: 设置环境变量

```bash
# 必须设置 OpenAI API Key
export OPENAI_API_KEY="sk-xxx你的key"

# 如果使用代理 (可选)
export OPENAI_BASE_URL="https://your-proxy.com/v1"
```

## Step 3: 启动应用

```bash
cd langchain4j-learn  # 回到项目根目录

# 编译并启动
mvn spring-boot:run -pl 11-knowledge-base/knowledge-base-service
```

启动成功后会看到:
```
Redis 向量存储初始化完成, host=localhost, port=6379
数据库知识源初始化完成
Started KnowledgeBaseApplication in 5.xxx seconds
```

## Step 4: 同步知识 (首次必须执行)

数据库里有示例数据，但还没有向量化。需要触发同步:

```bash
# 触发同步
curl -X POST "http://localhost:8080/api/knowledge/sync/trigger?source=database&mode=incremental"
```

成功响应:
```json
{
  "success": true,
  "docCount": 5,
  "segmentCount": 15,
  "message": "成功"
}
```

## Step 5: 测试搜索

```bash
# 搜索请假相关
curl "http://localhost:8080/api/knowledge/search?q=如何请假"

# 搜索报销相关
curl "http://localhost:8080/api/knowledge/search?q=费用报销怎么弄"

# 搜索代码规范
curl "http://localhost:8080/api/knowledge/search?q=变量怎么命名"
```

## API 列表

| 功能 | 方法 | URL | 参数 |
|------|------|-----|------|
| 搜索 | GET | /api/knowledge/search | q, limit, minScore, category |
| 触发同步 | POST | /api/knowledge/sync/trigger | source, mode |
| 同步状态 | GET | /api/knowledge/sync/status | - |
| 知识源列表 | GET | /api/knowledge/sync/sources | - |
| 健康检查 | GET | /api/knowledge/sync/health | - |

## 停止服务

```bash
# 停止 Spring Boot 应用: Ctrl+C

# 停止 Docker 容器
docker-compose down

# 如果要删除数据
docker-compose down -v
```
